Python

Python es un lenguaje de programación interpretado, de alto nivel y propósito general, creado por Guido van Rossum en 1991. Destaca por su legibilidad, simplicidad y versatilidad, lo que lo ha convertido en uno de los lenguajes más populares del mundo para diversos ámbitos de aplicación.

Características fundamentales

Sintaxis y filosofía

  • Legibilidad: Sintaxis clara e intuitiva con indentación significativa
  • Filosofía Zen: "Hermoso es mejor que feo, explícito es mejor que implícito"
  • Tipado dinámico: No requiere declaración de tipos de variables
  • Multipropósito: Adecuado para casi cualquier tipo de proyecto
  • Interpretado: No necesita compilación previa a ejecución
  • Fuerte comunidad: Gran base de usuarios y contribuidores

Características técnicas

  • Gestión automática de memoria: Recolector de basura
  • Funciones de primera clase: Soporte para programación funcional
  • Orientado a objetos: Todo es un objeto
  • Extensible: Integrable con C/C++
  • Gran biblioteca estándar: "Baterías incluidas"
  • Multiplataforma: Windows, macOS, Linux, y más

Ejemplo básico

# Un simple programa que muestra la elegancia de Python
def fibonacci(n):
    """Genera los primeros n números de la secuencia Fibonacci."""
    sequence = [0, 1]
    
    # Genera la secuencia
    for i in range(2, n):
        sequence.append(sequence[i-1] + sequence[i-2])
    
    return sequence

# Crear una lista de números Fibonacci
fib_numbers = fibonacci(10)

# Usar list comprehension para filtrar números pares
even_fibs = [num for num in fib_numbers if num % 2 == 0]

# Usar funciones de orden superior para calcular suma
total = sum(even_fibs)

print(f"Números Fibonacci: {fib_numbers}")
print(f"Números pares: {even_fibs}")
print(f"Suma de números pares: {total}")

Versiones y evolución

Python 2 vs Python 3

Python 3 fue lanzado en 2008 como una revisión importante que rompía la compatibilidad con Python 2. Desde el 1 de enero de 2020, Python 2 ya no recibe soporte oficial.

Característica Python 2 Python 3
Print print "Hola" print("Hola")
División de enteros 5 / 2 = 2 5 / 2 = 2.5
Unicode Strings ASCII por defecto Strings Unicode por defecto
Excepciones except Exception, e: except Exception as e:
Iteradores xrange(), range() Solo range() (optimizado)

Versiones principales de Python 3

  • Python 3.6 (2016): f-strings, variables anotadas, números asíncronos
  • Python 3.7 (2018): PostreSQL data class, mejoras en asyncio
  • Python 3.8 (2019): Operador walrus (:=), posicionamiento preciso de parámetros
  • Python 3.9 (2020): Operadores de fusión de diccionarios (|), funciones flexibles de cadenas
  • Python 3.10 (2021): Coincidencia de patrones, mejores mensajes de error
  • Python 3.11 (2022): Mejoras de rendimiento significativas, anotaciones de tipo más precisas
  • Python 3.12 (2023): Mejor rendimiento, f-strings mejoradas, nuevas APIs para subprocesos

from fastapi import FastAPI, HTTPException, Depends
from pydantic import BaseModel
from typing import List, Optional
import uvicorn

app = FastAPI(title="API de Productos")

# Modelo de datos
class Product(BaseModel):
    id: Optional[int] = None
    name: str
    price: float
    stock: int

# Base de datos simulada
db = []
counter = 0

@app.post("/products/", response_model=Product)
def create_product(product: Product):
    global counter
    counter += 1
    
    new_product = product.dict()
    new_product["id"] = counter
    db.append(new_product)
    
    return new_product

@app.get("/products/", response_model=List[Product])
def get_products():
    return db

@app.get("/products/{product_id}", response_model=Product)
def get_product(product_id: int):
    for product in db:
        if product["id"] == product_id:
            return product
    raise HTTPException(status_code=404, detail="Producto no encontrado")

if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8000)

import os
from pathlib import Path
import shutil
import datetime as dt

def organize_files(directory):
    """Organiza archivos por tipo en un directorio."""
    # Mapeo de extensiones a carpetas
    extensions = {
        'Documentos': ['.pdf', '.docx', '.txt'],
        'Imágenes': ['.jpg', '.png', '.gif'],
        'Videos': ['.mp4', '.mov', '.avi'],
        'Audio': ['.mp3', '.wav', '.flac']
    }
    
    # Crear directorio base
    base_dir = Path(directory)
    
    # Crear carpetas si no existen
    for folder in extensions:
        folder_path = base_dir / folder
        folder_path.mkdir(exist_ok=True)
    
    # Crear carpeta para otros tipos
    other_folder = base_dir / "Otros"
    other_folder.mkdir(exist_ok=True)
    
    # Organizar archivos
    files_moved = 0
    for file in base_dir.iterdir():
        if file.is_file():
            # Determinar destino
            destination = other_folder
            for folder, exts in extensions.items():
                if file.suffix.lower() in exts:
                    destination = base_dir / folder
                    break
            
            # Mover archivo
            if destination != file.parent:
                shutil.move(str(file), str(destination / file.name))
                files_moved += 1
    
    return files_moved

# Ejecutar la organización
if __name__ == "__main__":
    import sys
    
    if len(sys.argv) > 1:
        directory = sys.argv[1]
    else:
        directory = os.getcwd()
    
    count = organize_files(directory)
    print(f"Organizados {count} archivos en {directory}")

import tkinter as tk
from tkinter import messagebox, ttk

class SimpleApp:
    def __init__(self, root):
        self.root = root
        self.root.title("Mi Aplicación Python")
        self.root.geometry("400x300")
        
        # Variables
        self.name_var = tk.StringVar()
        self.task_var = tk.StringVar()
        self.tasks = []
        
        # Crear widgets
        self.create_widgets()
    
    def create_widgets(self):
        # Frame principal
        main_frame = ttk.Frame(self.root, padding="20")
        main_frame.pack(fill=tk.BOTH, expand=True)
        
        # Entrada de nombre
        ttk.Label(main_frame, text="Nombre:").grid(row=0, column=0, sticky=tk.W, pady=5)
        ttk.Entry(main_frame, textvariable=self.name_var).grid(row=0, column=1, sticky=tk.EW, pady=5)
        
        # Entrada de tarea
        ttk.Label(main_frame, text="Nueva tarea:").grid(row=1, column=0, sticky=tk.W, pady=5)
        ttk.Entry(main_frame, textvariable=self.task_var).grid(row=1, column=1, sticky=tk.EW, pady=5)
        ttk.Button(main_frame, text="Añadir", command=self.add_task).grid(row=1, column=2, padx=5, pady=5)
        
        # Lista de tareas
        ttk.Label(main_frame, text="Tareas:").grid(row=2, column=0, sticky=tk.W, pady=5)
        self.task_list = tk.Listbox(main_frame, height=10)
        self.task_list.grid(row=3, column=0, columnspan=3, sticky=tk.NSEW, pady=5)
        
        # Botones
        btn_frame = ttk.Frame(main_frame)
        btn_frame.grid(row=4, column=0, columnspan=3, pady=10)
        ttk.Button(btn_frame, text="Completar", command=self.complete_task).pack(side=tk.LEFT, padx=5)
        ttk.Button(btn_frame, text="Salir", command=self.root.destroy).pack(side=tk.LEFT, padx=5)
        
        # Configurar expansión
        main_frame.columnconfigure(1, weight=1)
        main_frame.rowconfigure(3, weight=1)
    
    def add_task(self):
        task = self.task_var.get()
        if task:
            self.tasks.append(task)
            self.task_list.insert(tk.END, task)
            self.task_var.set("")
        else:
            messagebox.showwarning("Advertencia", "Por favor ingrese una tarea.")
    
    def complete_task(self):
        selection = self.task_list.curselection()
        if selection:
            index = selection[0]
            task = self.tasks.pop(index)
            self.task_list.delete(index)
            messagebox.showinfo("Completado", f"Tarea completada: {task}")
            
            if self.name_var.get():
                messagebox.showinfo("Mensaje", f"¡Buen trabajo, {self.name_var.get()}!")
        else:
            messagebox.showwarning("Advertencia", "Por favor seleccione una tarea.")

# Iniciar aplicación
if __name__ == "__main__":
    root = tk.Tk()
    app = SimpleApp(root)
    root.mainloop()

Buenas prácticas y estilo

PEP 8 - Guía de estilo para Python

PEP 8 es la guía de estilo oficial para código Python, que promueve consistencia y legibilidad.

Principios generales:

  • Indentación: 4 espacios (no tabs)
  • Longitud de línea: Máximo 79 caracteres
  • Líneas en blanco: Separar funciones y clases con dos líneas
  • Importaciones: Una por línea, agrupadas
  • Espacios en blanco: Evitar excesos, pero usar para legibilidad
  • Comentarios: Actualizados, relevantes, completos

Convenciones de nombres:

  • funciones_y_variables: snake_case
  • ClassesYExcepciones: CamelCase
  • CONSTANTES: MAYÚSCULAS_CON_GUIONES
  • _variable_interna: Prefijo guion bajo
  • __variable_privada: Prefijo doble guion bajo
  • método_especial__: Doble guion bajo al inicio y final

Ejemplo de estilo PEP 8:

"""Módulo de ejemplo que sigue las convenciones PEP 8.

Este módulo demuestra cómo aplicar las convenciones de estilo de código
recomendadas en PEP 8, la guía de estilo para Python.
"""

import os
import sys
from typing import List, Dict, Optional, Union

# Constantes
MAX_SIZE = 100
DEFAULT_LEVEL = 'info'


class ConfigManager:
    """Clase para gestionar configuraciones."""
    
    def __init__(self, config_file: str):
        """Inicializa un gestor de configuración.
        
        Args:
            config_file: Ruta al archivo de configuración.
        """
        self.config_file = config_file
        self._settings = {}
        self.__private_data = None
    
    def load_config(self) -> Dict[str, Union[str, int, bool]]:
        """Carga la configuración desde el archivo.
        
        Returns:
            Diccionario con la configuración cargada.
            
        Raises:
            FileNotFoundError: Si el archivo no existe.
        """
        if not os.path.exists(self.config_file):
            raise FileNotFoundError(f"Archivo no encontrado: {self.config_file}")
        
        # Código para cargar configuración
        self._settings = {'debug': True, 'level': DEFAULT_LEVEL}
        return self._settings


def process_data(items: List[str], max_items: Optional[int] = None) -> List[str]:
    """Procesa una lista de elementos.
    
    Args:
        items: Lista de elementos a procesar.
        max_items: Número máximo de elementos a procesar.
            Si es None, procesa todos los elementos.
    
    Returns:
        Lista de elementos procesados.
    """
    if max_items is None:
        max_items = len(items)
    
    # Esto es una línea muy larga que debería dividirse para cumplir con el límite de 79 caracteres
    processed_items = [
        item.strip().lower()
        for item in items[:max_items]
        if item and not item.startswith('#')
    ]
    
    return processed_items


if __name__ == "__main__":
    # Ejemplo de uso del módulo
    config = ConfigManager("settings.ini")
    try:
        settings = config.load_config()
        print(f"Configuración cargada: {settings}")
    except FileNotFoundError as e:
        print(f"Error: {e}", file=sys.stderr)

Decoradores

Funciones que modifican el comportamiento de otras funciones.

import time
import functools

def timer(func):
    """Mide el tiempo de ejecución de una función."""
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print(f"{func.__name__} ejecutada en {end_time - start_time:.4f} segundos")
        return result
    return wrapper

@timer
def procesar_datos(items):
    """Simula procesamiento de datos."""
    time.sleep(1)  # Simular trabajo
    return [item * 2 for item in items]

# Uso
resultado = procesar_datos(range(10))
print(resultado)

Generadores

Funciones que generan valores bajo demanda, ahorrando memoria.

def fibonacci_generator(limit):
    """Generador de números Fibonacci hasta un límite."""
    a, b = 0, 1
    while a < limit:
        yield a
        a, b = b, a + b

# Uso - los valores se generan uno a uno
for num in fibonacci_generator(1000):
    print(num, end=' ')

# Comparación con lista (carga todo en memoria)
fibonacci_list = list(fibonacci_generator(1000))
print(fibonacci_list)

Context Managers

Para gestión de recursos y limpieza automática.

class MiContextManager:
    """Ejemplo de context manager personalizado."""
    
    def __init__(self, nombre):
        self.nombre = nombre
        
    def __enter__(self):
        print(f"Iniciando {self.nombre}")
        return self
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        print(f"Finalizando {self.nombre}")
        # Manejo de excepciones
        if exc_type:
            print(f"Ocurrió un error: {exc_val}")
            return True  # Suprimir excepción
            
    def procesar(self):
        print("Procesando datos...")

# Uso
with MiContextManager("Proceso") as cm:
    cm.procesar()
    
# Versión usando contextlib
from contextlib import contextmanager

@contextmanager
def tempfile(nombre):
    try:
        f = open(nombre, 'w')
        yield f
    finally:
        f.close()
        import os
        os.remove(nombre)
        
with tempfile("temp.txt") as f:
    f.write("Datos temporales")

Metaprogramación

Código que manipula código.

class ModelMeta(type):
    """Metaclase para crear modelos."""
    
    def __new__(mcs, name, bases, attrs):
        # Añadir métodos o atributos automáticamente
        attrs['table_name'] = name.lower()
        
        # Añadir un método a la clase
        def save(self):
            print(f"Guardando {self.__class__.__name__} en la tabla {self.table_name}")
        attrs['save'] = save
        
        return super().__new__(mcs, name, bases, attrs)

# Usar la metaclase
class User(metaclass=ModelMeta):
    def __init__(self, name, email):
        self.name = name
        self.email = email

# La clase User tiene automáticamente table_name y método save
user = User("Juan", "juan@ejemplo.com")
print(user.table_name)
user.save()
