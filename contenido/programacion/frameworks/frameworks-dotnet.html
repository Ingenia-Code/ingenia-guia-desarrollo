Frameworks de .NET

.NET es un ecosistema de desarrollo robusto y versátil, desarrollado por Microsoft, que incluye una amplia variedad de frameworks para diferentes propósitos, desde desarrollo web hasta aplicaciones móviles y de escritorio. Los frameworks más populares en el ecosistema de .NET incluyen ASP.NET Core, Entity Framework, Xamarin, y otros. A continuación, se presenta una guía detallada sobre estos frameworks, sus características principales, y cómo utilizarlos en proyectos modernos.

Frameworks para desarrollo web

ASP.NET Core

ASP.NET Core es un framework moderno y multiplataforma para construir aplicaciones web y APIs. Es la evolución de ASP.NET, diseñado para ser más rápido, modular y ligero.

Características principales:

  • Multiplataforma: Funciona en Windows, Linux y macOS.
  • Alto rendimiento: Diseñado para ser rápido y eficiente.
  • Inyección de dependencias: Soporte nativo para inyección de dependencias.
  • Middleware: Arquitectura basada en middleware para manejar solicitudes y respuestas.
  • Razor Pages: Simplifica la creación de páginas web con sintaxis Razor.

Ejemplo básico:

using Microsoft.AspNetCore.Builder;
using Microsoft.AspNetCore.Hosting;
using Microsoft.Extensions.DependencyInjection;
using Microsoft.Extensions.Hosting;

var builder = WebApplication.CreateBuilder(args);
var app = builder.Build();

app.MapGet("/", () => "Hello, ASP.NET Core!");

app.Run();

Blazor

Blazor es un framework para construir interfaces de usuario interactivas en el navegador usando C# en lugar de JavaScript.

Características principales:

  • WebAssembly: Ejecuta código C# en el navegador usando WebAssembly.
  • Componentes: Basado en componentes reutilizables.
  • Razor: Usa sintaxis Razor para definir componentes.
  • Multiplataforma: Puede ejecutarse en el servidor (Blazor Server) o en el cliente (Blazor WebAssembly).

Ejemplo básico:

@page "/counter"

<h3>Counter</h3>

<p>Current count: @currentCount</p>

<button class="btn btn-primary" @onclick="IncrementCount">Click me</button>

@code {
    private int currentCount = 0;

    private void IncrementCount()
    {
        currentCount++;
    }
}

Frameworks para acceso a datos

Entity Framework Core

Entity Framework Core (EF Core) es un ORM (Object-Relational Mapper) que permite a los desarrolladores trabajar con bases de datos usando objetos .NET en lugar de escribir consultas SQL manuales.

Características principales:

  • Multiplataforma: Funciona en Windows, Linux y macOS.
  • Code First: Permite definir el modelo de datos en código y generar la base de datos automáticamente.
  • Migrations: Facilita la gestión de cambios en el esquema de la base de datos.
  • Soporte para múltiples bases de datos: SQL Server, SQLite, PostgreSQL, MySQL, y más.

Ejemplo básico:

using Microsoft.EntityFrameworkCore;

public class BloggingContext : DbContext
{
    public DbSet<Blog> Blogs { get; set; }

    protected override void OnConfiguring(DbContextOptionsBuilder optionsBuilder)
    {
        optionsBuilder.UseSqlite("Data Source=blogging.db");
    }
}

public class Blog
{
    public int BlogId { get; set; }
    public string Url { get; set; }
}

Frameworks para desarrollo móvil

Xamarin

Xamarin es un framework para construir aplicaciones móviles nativas usando C# y .NET. Permite compartir código entre plataformas (iOS, Android, y Windows).

Características principales:

  • Multiplataforma: Código único para iOS, Android y Windows.
  • Acceso nativo: Acceso completo a APIs nativas de cada plataforma.
  • Xamarin.Forms: Permite crear interfaces de usuario compartidas entre plataformas.
  • Integración con Visual Studio: Herramientas de desarrollo integradas en Visual Studio.

Ejemplo básico:

using Xamarin.Forms;

public class App : Application
{
    public App()
    {
        MainPage = new ContentPage
        {
            Content = new Label
            {
                Text = "Hello, Xamarin!",
                VerticalOptions = LayoutOptions.Center,
                HorizontalOptions = LayoutOptions.Center
            }
        };
    }
}

Frameworks para desarrollo de escritorio

Windows Forms

Windows Forms es un framework para crear aplicaciones de escritorio para Windows. Es una de las tecnologías más antiguas de .NET, pero sigue siendo ampliamente utilizada.

Características principales:

  • Diseño visual: Permite diseñar interfaces de usuario arrastrando y soltando controles.
  • Eventos: Manejo de eventos para interactividad.
  • Compatibilidad: Funciona en todas las versiones de Windows.

Ejemplo básico:

using System.Windows.Forms;

public class Program : Form
{
    public Program()
    {
        var button = new Button
        {
            Text = "Click Me",
            Location = new System.Drawing.Point(100, 100)
        };

        button.Click += (sender, e) => MessageBox.Show("Hello, Windows Forms!");

        Controls.Add(button);
    }

    [STAThread]
    static void Main()
    {
        Application.Run(new Program());
    }
}
