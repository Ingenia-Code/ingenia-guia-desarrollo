Python

Python es un lenguaje de programación de alto nivel, interpretado y multiparadigma, creado por Guido van Rossum en 1991. Es conocido por su sintaxis clara y legible, lo que lo hace ideal para principiantes y profesionales. Python es ampliamente utilizado en desarrollo web, ciencia de datos, inteligencia artificial, automatización, y más.

Características principales

Fundamentos

  • Interpretado: No requiere compilación, se ejecuta directamente.
  • Dinámicamente tipado: Los tipos de datos se asignan en tiempo de ejecución.
  • Orientado a objetos: Soporta clases, herencia y polimorfismo.
  • Multiplataforma: Funciona en Windows, macOS, Linux y más.
  • Amplia biblioteca estándar: Incluye módulos para casi cualquier tarea.
  • Comunidad activa: Gran cantidad de recursos y bibliotecas de terceros.

Paradigmas soportados

  • Orientado a objetos: Clases, herencia, encapsulamiento.
  • Imperativo: Secuencias de declaraciones que cambian el estado.
  • Funcional: Soporta funciones de primera clase, lambdas y map/filter.
  • Procedural: Programación basada en procedimientos y funciones.

Evolución de Python

Historia y versiones principales

  • 1991: Lanzamiento de Python 0.9.0 por Guido van Rossum.
  • 2000: Python 2.0 introdujo list comprehensions y recolector de basura.
  • 2008: Python 3.0 rompió la compatibilidad con Python 2 para mejorar el lenguaje.
  • 2020: Fin del soporte para Python 2.7.
  • 2021: Python 3.10 introdujo pattern matching y mejoras en el sistema de tipos.
  • 2023: Python 3.11 mejoró el rendimiento y añadió nuevas características.

Características modernas (Python 3.10+)

Sintaxis y estructuras

# Pattern Matching (Python 3.10+)
def describe_value(value):
    match value:
        case 0:
            return "Cero"
        case [x, y]:
            return f"Lista con dos elementos: {x}, {y}"
        case _:
            return "Otro valor"

# Tipos de datos mejorados (Python 3.9+)
from typing import Union

def square(x: Union[int, float]) -> Union[int, float]:
    return x ** 2

Asincronía moderna

# Async/Await
import asyncio

async def fetch_data():
    await asyncio.sleep(1)
    return "Datos obtenidos"

async def main():
    result = await fetch_data()
    print(result)

asyncio.run(main())

Entornos de ejecución

CPython

La implementación oficial de Python, escrita en C. Es la más utilizada y la referencia para el lenguaje.

Características:

  • Implementación oficial y más extendida.
  • Incluye el intérprete estándar.
  • Soporte para extensiones en C.

PyPy

Una implementación alternativa de Python que utiliza un compilador JIT (Just-In-Time) para mejorar el rendimiento.

Características:

  • Mayor rendimiento que CPython en muchos casos.
  • Compatible con la mayoría del código Python.
  • Ideal para aplicaciones que requieren alto rendimiento.

Concurrencia con Threads

from threading import Thread

def print_numbers():
    for i in range(5):
        print(i)

thread = Thread(target=print_numbers)
thread.start()
thread.join()

Mejores prácticas

Arquitectura y organización de código

Patrones de diseño comunes

  • Singleton: Garantiza una única instancia de una clase.
  • Factory: Encapsula la creación de objetos.
  • Observer: Notifica cambios a múltiples observadores.

Estructura de proyectos

Performance

  • Uso de generadores: Ahorra memoria en iteraciones grandes.
  • Evitar bucles innecesarios: Usar list comprehensions y funciones built-in.
  • Optimización de consultas: Usar índices en bases de datos.
